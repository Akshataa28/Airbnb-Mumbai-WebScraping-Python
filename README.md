The project extracts publicly available Airbnb listing data for Mumbai using Python web scraping techniques and stores the processed data in CSV format for exploratory analysis.

## Project Objective
To extract key details of Airbnb listings in Mumbai such as pricing, location, ratings, and property details using web scraping techniques, and prepare a clean dataset for analysis.

## Tech Stack
- Python
- BeautifulSoup
- Requests
- Pandas
- Google Colab

## Dataset
<a href="https://github.com/Akshataa28/Airbnb-Mumbai-WebScraping-Python/blob/main/AirbnbMumbai.csv">Airbnb Mumbai Listings Dataset</a>

## Process
- Sent HTTP requests to fetch Airbnb web page content.
- Parsed HTML data using BeautifulSoup.
- Extracted relevant listing details.
- Cleaned and structured the data using Pandas.
- Exported the final dataset to CSV format.

## Key Features
- Automated data extraction using Python.
- Structured data storage in CSV format.
- Reusable and well-organized scraping logic.
- Suitable for exploratory data analysis and visualization.
